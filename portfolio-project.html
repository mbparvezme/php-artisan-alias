Project Overview

PHP Artisan Alias is a VS Code extension designed to streamline Laravel development workflows. It automatically detects Laravel projects and provides intelligent command aliases that reduce typing and improve developer productivity.

The extension transforms verbose `php artisan` commands into intuitive shortcuts, making Laravel development faster and more enjoyable. With 60+ pre-configured aliases, developers can focus on building features rather than typing long commands.

$ mkmodel User
$ migrate
$ serve
$ tinker

Instead of typing `php artisan make:model User`

Key Features

60+ Intelligent Aliases

Comprehensive set of aliases covering all common Laravel artisan commands, from make commands to database operations.

Auto-Detection

Automatically detects Laravel projects and sets up aliases when terminals are opened, requiring zero configuration.

Cross-Platform

Works seamlessly across Windows (PowerShell/CMD), macOS, and Linux terminals with intelligent shell detection.
